Группы доступа контрагентов

Справочник предназначен для описания доступа к элементам справочника Контрагенты по виду доступа Группы контрагентов. Справочник используется в том случае, если необходимо ограничить доступ пользователям к конкретным контрагентам. Информация о том, к какой группе доступа принадлежит тот или иной контрагент, указывается в справочнике Контрагенты. Предусмотрена возможность отнесения контрагентов к определенной группе доступа с помощью сервисного механизма группового изменения реквизитов, который можно вызвать для выделенных контрагентов из формы списка справочника Контрагенты.

Пример. Менеджеры по продажам работают с различными контрагентами. Одни менеджеры работают с оптовыми покупателями, а другие с розничными. Менеджерам должна быть доступна информация только по тем контрагентам, с которыми они работают и не должна быть доступна информация по контрагентам других менеджеров. В справочник Группы доступа контрагентов вводится информация о всех тех группах контрагентов, с которыми работают менеджеры: Оптовые покупатели и Розничные покупатели. Для каждого пользователя в качестве разрешенной группы доступа контрагентов назначается соответствующая группа доступа: Оптовые покупатели или Розничные покупатели.

Таким образом, для менеджеров будет ограничен список доступных им контрагентов:

  • менеджеры по работе с оптовыми покупателями будут видеть информацию только по тем контрагентам, для которых определена группа доступа контрагентов Оптовые покупатели
  • менеджеры по работе с розничными покупателями будут видеть информацию только по тем контрагентам, для которых определена группа доступа контрагентов Розничные покупатели


Важно. Установить ограничения по группам доступа контрагентов возможно только в том случае, если в разделе Настройки/Настройки пользователей и прав установлены опции Ограничивать доступ на уровне записей и Группы доступа контрагентов.
