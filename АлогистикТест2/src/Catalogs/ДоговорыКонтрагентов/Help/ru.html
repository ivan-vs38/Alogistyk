Справочник "Договоры контрагентов"

Справочник предназначен для хранения договоров расчетов, заключенных с контрагентами. Данный справочник подчинен справочнику "Контрагенты".

 

Список договоров с контрагентом определяется теми договорами, которые подчинены соответствующему элементу справочника "Контрагенты".

Для каждого контрагента можно выбрать т.н. договор по умолчанию, который будет подставляться по умолчанию в документы.

Аналитический учет расчетов с контрагентами ведется по каждому договору.

 

При создании договора заполняются Номер договора и Дата заключения.


Если включен учет по различным валютам, то так же заполняются следующие данные:

  • Валюта расчетов по договору с контрагентом. Выбирается из справочника "Валюты".
  • Расчеты в у.е. Признак расчетов по договору в условных единицах.


Для включения данной настройки необходимо отметить пункт в настройках:

Еще больше возможностей — Деньги — Несколько валют


Ценообразование

  • Вид цен, Вид скидки, Вид цен контрагента. Значения по умолчанию. Подставляются в документы при выборе договора.
  • Срок оплаты поставщику. Указывается срок отсрочки платежа по договору с поставщиком.
  • Срок оплаты от покупателя. Указывается срок отсрочки платежа по договору с покупателем.


Биллинг


Данный блок доступен при включении соответствующей настройки:

Еще больше возможностей — Продажи — Биллинг и работа с регулярными услугами

Договор обслуживания — такой договор с контрагентом, который подразумевает оказание регулярных услуг или работ клиенту с определенной периодичностью.

Для оформления договора обслуживания необходимо в первую очередь поставить соответствующий признак Это договор обслуживания и заполнить следующие поля:

  • Период действия
    Дата начала и окончания действия договора. 
    Для оформления бессрочного договора можно не заполнять дату окончания.
  • Периодичность
    Определяет частоту оказания регулярных услуг и выставления счетов.
  • Тарифный план
    Определяет набор и порядок предоставления услуг по договору обслуживания.
    Подробнее см. справочник Тарифные планы договоров обслуживания


Процесс выставления счетов по договорам обслуживания автоматизирован, что позволяет не вспоминать когда и кому и, главное, за что выставлять счета!
Подробнее см. Выставление счетов по договорам обслуживания.


Помимо этого, есть возможность учитывать доходы и расходы по договорам обслуживания. Так, для каждого договора обслуживания программа будет автоматически создавать уникальное направление деятельности, которое нельзя перевыбрать вручную.
Для этого нужно включить настройку:

Еще больше возможностей — Продажи — Вести учет расходов в разрезе договоров обслуживания


Обмен данными

  • Организация. Указывается организация, с которой заключен договор. Выбирается из справочника "Организации".
  • Вид договора с контрагентом: с поставщиком, с покупателем, с комитентом, с комиссионером или прочее. Реквизит не является обязательным к заполнению и используется только при обмене данными с конфигурацией "Бухгалтерия предприятия".


Печать договора


На вкладке "Текст договора производится формирование печатной формы и, соответственно, печать договора. Для этого необходимо выбрать бланк договора, на основе которого печатная форма будет сформирована, и заполнить необходимые поля.
