Отчет "Воронка продаж"

Наглядно отображает распределение заказов по стадиям процесса продаж. Формируется на основании истории изменения состояний заказов покупателей и заказ-нарядов.

Колонки отчета:

  • Состояние - состояние заказа, как оно настроено в справочнике Вид заказа.
  • Кол-во - количество заказов, прошедших через текущее состояние или находящихся в нём. Заказы, перепрыгнувшие через состояние, считаются побывавшими в этом состоянии. Шкала отображает соотношение к заказам, побывавшим в самом первом состоянии.
  • Конверсия - процент заказов перешедших из предыдущего состояния.
  • t, ч - среднее время, проведенное заказами в текущем состоянии.
  • Сумма - сумма заказов, прошедших через текущее состояние.
  • Потери - количество отмененных в текущем состоянии заказов . Шкала пропорционально показывает распределение отмен по причинам. Процент потерь рассчитывается от количества заказов побывавших в текущем состоянии.

 

Варианты отчета:

  • По заказам - отчет формируется с группировкой по виду заказа.
  • По покупателям - отчет формируется с группировкой по контрагенту в заказе. Для формирования требуется задать фильтр по покупателям или группам покупателей.
  • По менеджерам - отчет формируется с группировкой по ответственному в заказе. Для формирования требуется задать фильтр по сотрудникам или группам сотрудников.
  • По подразделениям - отчет формируется с группировкой по подразделению. Для формирования требуется задать фильтр по подразделениям.
