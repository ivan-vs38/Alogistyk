Документ "Расходная накладная"

Документ предназначен для хранения информации по отражению в учете различных операции по реализации (отгрузке) запасов и услуг стороннему контрагенту.

В зависимости от установленного вида операции с помощью данного документа может быть зафиксирована реализация (отгрузка) запасов и услуг по различным операциям:

  • Продажа покупателю,

  • Передача на комиссию,

  • Передача в переработку,

  • Передача на ответхранение,

  • Возврат поставщику,

  • Возврат комитенту,

  • Возврат из переработки,

  • Возврат с ответхранения.

Необходимо заполнить следующие реквизиты:

  • Организация. В данном реквизите указывается собственная организация, от имени которой оформляется документ. Выбирается из справочника "Организации". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по нескольким организациям.
  • Вид операции. Указывается вид операции документа.
  • Контрагент. В данном реквизите указывается информация о контрагенте, которому реализуются (отгружаются) запасы и услуги. Выбирается из справочника "Контрагенты".

  • Договор. В данном реквизите указывается договор контрагента, в рамках которого оформляется документ. Выбирается из справочника "Договоры контрагентов".

  • Основание. В данном реквизите указывается "Приходная накладная" или "Счет на оплату" на основание которых сформирован данный документ. Реквизит обязателен для заполнения только для операций "Возврат поставщику", "Возврат комитенту" или "Возврат с ответхранения".

  • Заказ. В данном реквизите указывается заказ, по которому реализуются (отгружаются) запасы и услуги. Выбирается документ "Заказ покупателя", если вид операции документа "Продажа покупателю", "Передача на комиссию" или "Возврат из переработки". Выбирается документ "Заказ поставщику", если вид операции документа "Возврат поставщику", "Возврат комитенту" или "Передача в переработку". Реквизит отображается на форме, если в настройках параметров учета включено заказ в документах отгрузки устанавливается для вссего документа. Реквизит не обязателен для заполнения. Примечание: если вид операции документа "Передача на ответхранение" или "Возврат с ответхранения", реквизит не отображается на форме.

  • Склад. Структурная единица, с которой списываются запасы. Выбирается из справочника "Структурные единицы" с типом "Склад".

  • Ячейка. В данном реквизите указывается ячейка структурной единицы (стеллаж, участок), с которой списываются запасы. Выбирается из справочника "Ячейки". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по ячеистым складам и склад не имеет признак ордерного.

На форме документа расположена гиперссылка "Редактировать цены и валюту", по которой вызывается диалог редактирования видов цен, скидок и валюты данного документа.

В подвале формы можно указать ручную скидку на документ, процентом или суммой.

  • Запасы и услуги. Данная табличная часть может быть заполнена с помощью подбора (кнопка "Подбор" командной панели табличной части) либо методом ручного ввода. Реквизиты табличной части:

    • Номенклатура - позиция номенклатуры, расход которой отражается данным документом. Выбирается из справочника "Номенклатура" с типом "Запас" или "Услуга".

    • Характеристика - при необходимости в данном поле указывается характеристика номенклатуры. Выбирается из справочника "Характеристики номенклатуры". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по характеристикам номенклатуры.

    • Партия - в данном поле указывается партия номенклатуры. Выбирается из справочника "Партии номенклатуры". Реквизит обязателен для заполнения, если вид операции документа: "Возврат комитенту", "Возврат из переработки" или "Возврат с ответхранения". Реквизит отображается на форме, если в настройках параметров учета включено ведение партионного учета.

    • Количество - количество номенклатурной позиции, подлежащей реализации (отгрузке).

    • Резерв - количество зарезервированных запасов.

    • Ед. - единица измерения номенклатурной позиции. Выбирается из справочника "Классификатор единиц измерения" или "Единицы измерения".

    • Цена - цена реализуемой номенклатурной позиции.

    • % Скидки - процент ручной скидки (наценки).

    • % Авт. - процент автоматической скидки (наценки). Процент автоматической скидки редактируется в справочнике "Виды автоматических скидок и наценок". Расчёт автоматических скидок в документе осуществляется при записи документа и при нажатии кнопки "% Авт." над табличной частью.

      Сумма - стоимость реализуемой номенклатуры по данной строке.

    • Ставка НДС - ставка налога на добавленную стоимость. Выбирается из справочника "Ставки НДС".

    • Сумма НДС - сумма налога на добавленную стоимость по данной строке.

    • Всего - поле, автоматически заполняемое суммарной стоимостью запасов и услуг согласно настройкам цен и валют. Настройки устанавливаются в диалоге, вызываемом по гиперссылке "Редактировать цены и валюту".

    • Заказ - указывается заказ, по которому реализуются (отгружаются) запасы и услуги. Выбирается документ "Заказ покупателя", если вид операции документа "Продажа покупателю", "Передача на комиссию" или "Возврат из переработки". Выбирается документ "Заказ поставщику", если вид операции документа "Возврат поставщику", "Возврат комитенту" или "Передача в переработку". Реквизит отображается на форме, если в настройках параметров учета включено Заказ в документах отгрузки устанавливается для каждой строки отдельно. Реквизит не обязателен для заполнения. Примечание: если вид операции документа "Передача на ответхранение" или "Возврат с ответхранения", реквизит не отображается на форме.
    • Содержание - представление запаса в печатной форме. Чтобы реквизит стал видимым, необходимо выполнить "Все действия" - "Изменить форму" и установить флаг напротив реквизита.
  • Доставка. Группа реквизитов:

    • Услуга - номенклатура с типом "Услуга", использующаяся для отражения доставки в учете. Обязательный реквизит при использовании способа доставки отличного от "Самовывоз". Выбирается из справочника "Номенклатура".

    • Стоимость - стоимость доставки заказа.

    • % НДС, сумма НДС - данные НДС доставки.

    • Объем - общий объем заказа. Рассчитывается автоматически по объему элементов из табличной части "Запасы". Может быть отредактирован вручную.

    • Вес - вес заказа. Рассчитывается автоматически по весу элементов из табличной части "Запасы". Может быть отредактирован вручную.

  • Зачет предоплаты. Данная табличная часть заполняется сведениями по зачету авансов от покупателя или по зачету авансов поставщику, если документом отражается операция возврата. На форме документа расположена гиперссылка "Редактировать зачет предоплаты", по которой вызывается диалог редактирования табличной части "Зачет предоплаты". Реквизиты табличной части:

    • Документ - документ поступления авансов: "Расход со счета", "Расход из кассы", "Поступление в кассу", "Поступление на счет".

    • Сумма расчетов - сумма в валюте договора.

    • Курс - курс пересчета в валюту документа.

    • Кратность - кратность пересчета в валюту документа.

    • Сумма платежа - указывается в валюте документа.

  • Дополнительная информация. Группа реквизитов:

    • Подразделение - информация о подразделении-исполнителе. Выбирается из справочника "Структурные единицы" с типом "Административно-управленческое подразделение".

    • Ответственный - информация о сотруднике, ответственном за ввод текущего документа. Выбирается из справочника "Сотрудники". Реквизит необязателен для заполнения.

    • Реквизиты печати - дополнительные сведения, которые используются при заполнении печатных форм.

Документ "Расходная накладная" может быть введен на основании документов:

  • "Приходная накладная",
  • "Счет на оплату",
  • "Заказ покупателя",
  • "Заказ поставщику".
