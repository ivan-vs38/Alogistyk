Товарно-транспортная накладная ЕГАИС (исходящая)

Документ предназначен для регистрации в системе отправленных в ЕГАИС товарно-транспортных накладных.

При оформлении нового документа доступен сервис автоматического подбора справок 2 для указанной в табличной части Товары алкогольной продукции.

На каждую отправленную ТТН должен быть получен один из актов: Акт подтверждения, Акт отказа или Акт расхождений. 

Если у грузоотправителя установлен признак учета остатков, то при получении акта подтверждения или акта расхождений, документ сформирует расход по регистру Остатки алкогольной продукции ЕГАИС.

Доступность документа определяется функциональной опцией Вести сведения для деклараций по алкогольной продукции.
