Акт постановки на баланс ЕГАИС

Документ предназначен для выгрузки в ЕГАИС информации о принимаемой на баланс алкогольной продукции.

Документ позволяет оприходовать продукцию на склад или в торговый зал. Для этого предназначен реквизит Вариант постановки на баланс. При постановке на склад требуется заполнить реквизиты табличной части: Количество по справке 1, Номер ТТН, Дата ТТН, Дата розлива продукции.

В реквизите Причина постановки на баланс требуется выбрать причину, на основании которой формируется акт. Если причиной является Пересортица, то нужно обязательно указать Акт списания ЕГАИС, которым списываются излишки по пересортице.

Если у организации установлен признак учета остатков, то документ в статусе Передан в ЕГАИС формирует приход по регистру Остатки алкогольной продукции ЕГАИС.

Доступность документа определяется функциональной опцией Вести сведения для деклараций по алкогольной продукции.
