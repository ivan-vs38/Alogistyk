Акт списания ЕГАИС

Документ предназначен для выгрузки в ЕГАИС информации о списываемой алкогольной продукции.

Документ позволяет списывать продукцию со склада или из торгового зала. Для этого предназначен реквизит Вариант списания. При списании со склада требуется обязательно указать справку 2, по которой товар был принят.

В реквизите Причина списания требуется выбрать причину, на основании которой формируется акт. Причины списания Иные цели и Реализация доступны только при списании из торгового зала или при использовании формата обмена V2.

Если у организации установлен признак учета остатков, то документ в статусе Передан в ЕГАИС формирует расход по регистру Остатки алкогольной продукции ЕГАИС.

Доступность документа определяется функциональной опцией Вести сведения для деклараций по алкогольной продукции.
