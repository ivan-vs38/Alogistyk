Печать этикеток и ценников

Обработка предназначена для печати этикеток и ценников на товары. Ценники и этикетки печатаются в соответствии с шаблонами, которые можно добавлять и редактировать. 

Создание (редактирование) шаблонов этикеток и ценников осуществляется по кнопке Шаблоны этикеток и ценников.

Для печати этикеток и ценников при помощи команды Показать отбор можно отобрать товары по различным критериям (по позициям номенклатуры, по группе номенклатуры, по наличию товара на складе и т.д.). При помощи команды Показать режим заполнения отображаются дополнительные данные о режимах заполнения этикеток и ценников (при условии выбора режима печати Печать этикеток и ценников).

Печать осуществляется по виду цен, который требуется указать в соответствующем поле.

При помощи команды Установить для выделенных строк можно назначить для выделенных строк печатаемое количество и шаблон этикетки или ценника.

Печать этикеток и ценников может быть осуществлена при помощи соответствующей команды из документа Приходная накладная.

Отметку выбора для печати можно установить только для тех товаров, для которых определен хотя бы один шаблон для печати ценника или этикетки и установлено печатаемое количество.
